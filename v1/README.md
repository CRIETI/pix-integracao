# V1

Esse projeto é mais simples, não há separação por controllers.

## Como executar

1. Modificar o arquivo de conexão com o banco de dados
```
vim db/index.ts
```

2. Instalar os pacotes
```
npm install
```

3. Executar o projeto
```
ts-node index.ts
```

4. Abrir o arquivo `index.js` no navegador
